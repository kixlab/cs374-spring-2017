Activity: Ordering Effects

We wish to conduct an experiment to answer the following question: Do people type dictionary words faster than they do gibberish?

1. Find a partner

  • One of you is user 1, the other is user 2. User 1 goes first.
  • Depending on your section, run the appropriate experiment with your partner.
  • Use one computer to input your data.

2. Load Your Section's Game

  • Open the game in a separate tab in your browser.

3. Answer thought questions about your condition:

  • Section A: Between Subjects

    This game is between subjects. One of you was in one condition (typing dictionary words), and the other person was in a different condition (typing gibberish). Think about the advantages and disadvantages of this set up. Is it appropriate for this particular experiment? When might it be important to pick a between subjects experiment?

  • Section B: Within Subjects, fixed ordering

    This game is within subjects. Each of you were in both conditions, and you both did the dictionary condition first, and then the gibberish condition. Since you both did this in the same order, this is not counterbalanced. Think about the advantages and disadvantages of this set up. Is it appropriate for this particular experiment? When might it be important to pick a within subjects experiment? When might you want or not want counterbalancing?

  • Section C: Within Subjects, randomized ordering

    This game is within subjects. Each of you were in both conditions, and you were randomly shown words in each condition. Think about the advantages and disadvantages of this set up. Is it appropriate for this particular experiment? When might it be important to pick a within subjects experiment? When might you want or not want randomization of conditions?

4. Answer these questions:

  • What were the independent variables and dependent variables in this experiment?
  • What about other variables that might affect the experiment -- which were you controlling for, and which were uncontrolled? What differences between each pair of people might threaten internal validity?
  • What are the ordering effects that might come into play in this task?
  • What biases in this experiment might be caused by your expectations about how it should turn out?
