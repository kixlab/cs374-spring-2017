17: Improving Readability

In this activity, we'll improve the readability of some text by adjusting the font of the body.

CSS properties

We'll use CSS to fine tune the appearance of our typefaces. A reference is available on MDN or just Google for "CSS font", but some properties you may find useful are:

  • font-style — example values: normal, italic, oblique
  • font-family — example values: "arial", "times", "sans-serif"
  • font-size — example values: 2em, 20px
  • line-height — distance between successive baselines, example values: 30px, 1.7em
  • word-spacing, letter-spacing — control the spacing between words, letters (you will probably never need these)

Some properties related to readability but not the font are:

  • text-indent — amount to indent the first line of text, example value: 2em
  • text-align — example values: left, justify
  • width — control the length of lines (long lines can be hard to follow across the screen)

Include Your Own Fonts

You might specify a font that the user doesn't have installed on his/her system. In this case, the browser will fall back to another font in the same family but your page might not look as intended.

One way to solve this problem is to use the CSS3 @font-face rule. This allows you to include your own TrueType or OpenType fonts in your page (except for IE9 which only allows embedded OpenType).

Another way is to use a Google hosted font. Try browsing through for a font you might want to use, then click the "Quick Use" link to see how to use the font in your page.
